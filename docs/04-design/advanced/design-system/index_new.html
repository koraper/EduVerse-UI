11. 접근성

색상 대비

  • • 일반 텍스트: 4.5:1 이상
  • • 큰 텍스트 (18px+): 3:1 이상
  • • UI 컴포넌트: 3:1 이상
  • • WCAG AA 기준 준수

키보드 네비게이션

  • • Tab 키로 모든 요소 접근 가능
  • • 포커스 시각적 표시
  • • 모달 포커스 트랩
  • • ESC 키로 닫기 지원

ARIA 속성

  • • 역할(role) 정의
  • • 상태(aria-pressed) 표시
  • • 라이브 영역 알림
  • • 레이블 제공

스크린 리더

  • • 숨겨진 라벨 제공
  • • 설명 텍스트 연결
  • • 의미있는 대체 텍스트
  • • 논리적 읽기 순서
