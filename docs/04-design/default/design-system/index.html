리스트와 튜플의 차이가 정확히 뭔가요?

📝 김교수:

리스트는 변경 가능(mutable)하고 튜플은 변경 불가능(immutable)합니다. 리스트는 []로, 튜플은 ()로 표현합니다.

모달 제목

이것은 EduVerse 디자인 시스템의 모달 컴포넌트입니다.

모달은 사용자의 주의를 집중시키거나 중요한 정보를 표시할 때 사용됩니다.

주요 특징:

  • 배경 오버레이 (bg-black/70)
  • 중앙 정렬
  • 반응형 크기 (max-w-2xl)
  • 스크롤 가능한 본문
  • 애니메이션 효과
