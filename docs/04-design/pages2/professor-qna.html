문자열을 숫자로 변환하는 방법

"123"이라는 문자열을 숫자 123으로 변환하려면 어떻게 해야 하나요? Number()와 parseInt()의 차이도 궁금합니다.

홍
홍길동 교수
1일 전

좋은 질문이에요! 문자열을 숫자로 변환하는 방법은 여러 가지가 있습니다:

  • Number(): 전체 문자열을 숫자로 변환합니다. "123.45"는 123.45가 됩니다.
  • parseInt(): 정수 부분만 추출합니다. "123.45"는 123이 됩니다.
  • parseFloat(): 소수점을 포함한 숫자로 변환합니다.
let str = "123";
let num1 = Number(str); // 123
let num2 = parseInt(str); // 123
let str2 = "123.45";
let num3 = Number(str2); // 123.45
let num4 = parseInt(str2); // 123

추가로 궁금한 점이 있으면 언제든 질문해주세요!

Flexbox의 justify-content와 align-items의 차이

두 속성이 모두 정렬과 관련된 것 같은데, 정확히 어떤 차이가 있나요?

홍
홍길동 교수
2일 전

Flexbox에서 두 속성의 차이는 정렬 방향입니다:

  • justify-content: 주축(main axis) 방향으로 정렬합니다. flex-direction이 row면 가로 방향입니다.
  • align-items: 교차축(cross axis) 방향으로 정렬합니다. flex-direction이 row면 세로 방향입니다.

쉽게 말하면, justify-content는 좌우 정렬, align-items는 상하 정렬이라고 생각하면 됩니다 (flex-direction: row일 때).

div와 span의 차이점이 무엇인가요?

둘 다 컨테이너 역할을 하는 것 같은데 언제 div를 쓰고 언제 span을 써야 하나요?

홍
홍길동 교수
3일 전

중요한 차이점은 display 속성입니다:

  • div: block 요소입니다. 새로운 줄에서 시작하고 전체 너비를 차지합니다.
  • span: inline 요소입니다. 줄 안에서 필요한 만큼만 공간을 차지합니다.
