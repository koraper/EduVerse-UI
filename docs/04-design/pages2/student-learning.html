🎯 미션: Hello World!

사용자의 이름을 입력받아 "Hello, [이름]!" 형식으로 인사하는 프로그램을 작성하세요.

요구사항:

  • • 표준 입력으로 이름 받기
  • • "Hello, " + 이름 + "!" 형식으로 출력
  • • 대소문자 구분 없음
# 여기에 코드를 작성하세요

name = input()
print(f"Hello, {name}!")
// 실행 결과
