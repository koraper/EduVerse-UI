## 1. 개요

### 1.1 목적
- 오프라인 수업 중 학생들을 실시간으로 모니터링하고 지원
- 수업 후 학습 데이터 분석 및 개선점 도출

### 1.2 핵심 기능
- **실시간 모니터링**: 수업 중 모든 학생 진도/상태 한눈에 파악
- **Code Peek**: 막힌 학생의 코드 즉시 확인
- **학습 분석**: 수업 후 데이터 기반 개선점 도출

## 8. 반응형 레이아웃

### 8.1 Desktop (1920px+)
- 실시간 모니터링: 6-column grid
- Code Peek: 모달 (max-w-4xl)

### 8.2 Laptop (1024px ~ 1919px)
- 실시간 모니터링: 4-column grid
- Code Peek: 모달 (max-w-3xl)

### 8.3 Tablet (768px ~ 1023px)
- 실시간 모니터링: 3-column grid 또는 리스트 뷰
- Code Peek: 전체 화면

### 8.4 Mobile (< 768px)
- 실시간 모니터링: 리스트 뷰 (카드 1-column)
- Code Peek: 전체 화면
