## 1. 개요

### 1.1 목적
- 학생이 시나리오 기반 학습을 수행하고 진도를 관리하는 모든 화면
- 오프라인 수업 중 학생이 코드를 작성하고 실행하며 교수자와 소통

### 1.2 핵심 화면
- **시나리오 학습 페이지**: 2단 레이아웃 (시나리오 + 코드 에디터)
- **진도 확인 대시보드**: 주차별 진행 상황 시각화
- **초대 코드 입력**: 수업 참여 첫 단계

### 1.3 관련 문서
- [기능 명세서 Part 2](../../03-specs/02-student-features.md)
- [디자인 시스템](../01-eduverse-design-system.md)

┌───────────────────────────────────────────────────────────┐

│                    수업에 참여하기                          │

│  선생님이 알려준 6자리 초대 코드를 입력해주세요             │

│                                                            │
│                                                            │
│  초대 코드 *                                               │
│  ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐ ┌───┐                    │
│  │ A │ │ B │ │ C │ │ 1 │ │ 2 │ │ 3 │                    │
│  └───┘ └───┘ └───┘ └───┘ └───┘ └───┘                    │
│  (각 칸: w-12 h-14 text-2xl text-center)                 │
│  (자동으로 다음 칸 이동, 대문자 자동 변환)                 │
│                                                            │
│                                                            │
│  예시: ABC123                                              │
│  (text-gray-500, text-sm)                                │
│                                                            │
│                                                            │
│  ┌──────────────────────────────────────────────────────┐│
│  │                 참여하기                              ││
│  │  (bg-indigo-600, hover:bg-indigo-700, py-3)         ││
│  └──────────────────────────────────────────────────────┘│
│                                                            │
│                                                            │
│  [취소]                                                    │
│  (text-gray-400, hover:text-white)                       │
│                                                            │
└───────────────────────────────────────────────────────────┘

**기능**:
- 자동 Focus: 첫 번째 칸에 자동 포커스
- 자동 이동: 입력 시 다음 칸으로 자동 이동
- 붙여넣기: "ABC123" 붙여넣기 시 자동 분할
- 대문자 변환: 소문자 입력 시 자동 대문자 변환
- Backspace: 이전 칸으로 이동
```

### 8.2 초대 코드 입력 (6칸)

```tsx
// Component: InviteCodeInput.tsx
export const InviteCodeInput: React.FC = () => {
  const [code, setCode] = useState(['', '', '', '', '', '']);
  const inputRefs = useRef<(HTMLInputElement | null)[]>([]);

  const handleChange = (index: number, value: string) => {
    const newCode = [...code];
    newCode[index] = value.toUpperCase();
    setCode(newCode);

    // 자동으로 다음 칸 이동
    if (value && index < 5) {
      inputRefs.current[index + 1]?.focus();
    }
  };

  const handleKeyDown = (index: number, e: React.KeyboardEvent) => {
    if (e.key === 'Backspace' && !code[index] && index > 0) {
      inputRefs.current[index - 1]?.focus();
    }
  };

  const handlePaste = (e: React.ClipboardEvent) => {
    e.preventDefault();
    const pastedText = e.clipboardData.getData('text').toUpperCase();
    const newCode = pastedText.slice(0, 6).split('');
    setCode([...newCode, ...Array(6 - newCode.length).fill('')]);
  };

  return (
    <div className="flex space-x-2 justify-center">
      {code.map((digit, index) => (
        <input
          key={index}
          ref={el => inputRefs.current[index] = el}
          type="text"
          maxLength={1}
          value={digit}
          onChange={e => handleChange(index, e.target.value)}
          onKeyDown={e => handleKeyDown(index, e)}
          onPaste={index === 0 ? handlePaste : undefined}
          className="w-12 h-14 text-2xl text-center bg-gray-900 border-2
                     border-gray-700 rounded-lg focus:border-indigo-500
                     text-white font-mono"
        />
      ))}
    </div>
  );
};
```
