=== EDUVERSE UI DESIGN SYSTEM ANALYSIS ===

DESIGN SYSTEM OVERVIEW
=====================

EduVerse implements a comprehensive design system using:
- Tailwind CSS 3.4 with custom color palette
- React Context API for theme/font management
- Full dark/light/system theme support
- Mobile-first responsive design

KEY FINDINGS:

1. THEME SYSTEM
   - Three modes: light, dark, system (default)
   - Persistence via localStorage['selectedTheme']
   - Usage: useTheme() hook with conditional Tailwind classes

2. COLOR PALETTE (tailwind.config.js)
   - primary (cyan): #0ea5e9 at 500, 50-900 variants
   - secondary (slate): neutral grays for secondary elements
   - success (emerald): green for positive states
   - warning (amber): yellow for warnings
   - error (red): red for errors
   - info: #3b82f6 for info messages

3. RESPONSIVE DESIGN
   - sm: 640px (mobile to tablet)
   - md: 768px (tablet)
   - lg: 1024px (desktop)
   - Common pattern: grid grid-cols-1 sm:grid-cols-2 gap-4

4. CORE COMPONENTS
   - Button: variants (primary, secondary, success, etc), sizes (sm, md, lg)
   - Input: theme-aware, password toggle, icon support, length counter
   - Card: header/body/footer sections, padding options, hoverable
   - Badge: 8 color variants, light/dark mode aware, pill shape option
   - Modal: 5 size options, portal rendering, keyboard support
   - Toast: 4 variants, auto-dismiss, action buttons

5. TYPOGRAPHY
   - Fonts: Noto Sans KR (default), Poppins, Nanum Pen, Fira Code
   - Font switching via FontContext
   - Responsive text sizing: text-2xl sm:text-3xl pattern

6. SPACING SYSTEM
   - 4px base unit (p-4 = 16px, p-6 = 24px, p-8 = 32px)
   - Section spacing: space-y-6 (24px)
   - Component gaps: gap-2, gap-3, gap-4, gap-6

7. ANIMATIONS & TRANSITIONS
   - Standard: transition-colors duration-200/300
   - Hover: hover:bg-primary-700, hover:shadow-lg
   - Focus: focus:ring-2 focus:ring-offset-2 focus:ring-primary-500
   - Custom: blob animation (7s infinite)

CRITICAL FILES:
- tailwind.config.js (color palette, animations)
- src/contexts/ThemeContext.tsx
- src/contexts/FontContext.tsx
- src/components/common/ (Button, Input, Card, Badge, Modal, Toast)
- src/components/layout/Header.tsx (theme toggle example)
- src/pages/auth/LoginPage.tsx (auth page patterns)
- src/pages/student/StudentDashboardPage.tsx (dashboard patterns)

DESIGN SYSTEM MATURITY: Production-Ready

Strengths:
+ Comprehensive semantic color system
+ Full dark/light/system theme support
+ Mobile-first responsive design
+ Accessible focus states
+ Consistent spacing and typography
